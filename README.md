# CatFeeder


## Setup
1. Start the Pico W in Boot mode.
2. Flash the .uf2 firmware to the Pico W.
3. Copy `~/Source/CatFeeder.py` to the root directory of the Pico W as `code.py`.
4. Edit the `~/Source/settings.template.toml` and save it as `settings.toml` in the root directory of the Pico W.
5. Copy `~/lib` to the root directory of the Pico W.
